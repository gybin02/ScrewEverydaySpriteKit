# 场景覆盖层与奖励弹窗规格 (overlay_popups)

本文档描述游戏通关胜利/失败的场景覆盖层，以及在游戏关卡结束结算时弹出的 Daily 任务达成和图鉴获得弹窗规范。

---

## 1. 胜利与失败场景覆盖层

胜利与失败界面以半透明覆盖层（`overlayLayer`）的形式叠加在当前游戏场景最上方。

* **遮罩底板**：全屏矩形，填充黑色（72% 透明度）。
* **点击行为**：点击屏幕任意位置可重开新关卡。

### 1.1 胜利覆盖层 (Victory Overlay)
* **标题**：屏幕中心上方 `60` 逻辑点。使用 `AvenirNext-Bold`，字号 `72`，绿色 `#2A9D8F`。文案为 `通关!`。
* **提示**：屏幕中心下方 `40` 逻辑点。使用 `AvenirNext-Medium`，字号 `28`，白色。文案为 `点击屏幕重开（新关卡）`。
* **触发条件**：
  1. 所有板子和螺丝被消完，工具箱被清空。
  2. 备选区为空。
  3. 游戏仍处于 `playing`。

### 1.2 失败覆盖层 (Failure Overlay)
* **标题**：屏幕中心上方 `60` 逻辑点。使用 `AvenirNext-Bold`，字号 `72`，红色 `#E63946`。文案为 `失败`。
* **提示**：屏幕中心下方 `40` 逻辑点。使用 `AvenirNext-Medium`，字号 `28`，白色。文案为 `点击屏幕重开（新关卡）`。
* **触发条件**：
  1. 备选区 5 个孔已被装满。
  2. 玩家点击下一颗螺丝，由于没有同色工具箱，被迫只能飞入备选区。

---

## 2. 奖励与成就弹窗系统 (Popups)

这些弹窗在关卡结算界面展示。它们基于 `Block Blast` 中提取的精美美术素材进行拼接与还原。

### 2.1 弹窗通用基础结构

```
+---------------------------------------------+
|              [ window_btn_close.png ] (右上)|
|                                             |
|          [ bg_TittleUp.png ] (标题装饰)     |
|          ------------------------------     |
|                                             |
|          [ window_blue.png ] (大圆角蓝框)    |
|                                             |
|                                             |
|                                             |
|                                             |
|          [ main_btn_green.png ] (底部确认)  |
+---------------------------------------------+
```

* **背景底色**：全屏黑色半透明覆盖层（`die_mask.png` / 75% 透明度）。
* **弹窗背景底板**：使用 `resources/textures/achievement/window_blue.png`（大号蓝色三维圆角矩形）。
  * 逻辑尺寸：`500 × 640`。
  * 位置：屏幕正中央 `(360, 640)`。
* **顶部标题条装饰**：使用 `resources/textures/achievement/bg_TittleUp.png` 放置在底板顶部，渲染主标题文本。
* **关闭交互**：右上角放置圆形关闭按钮 `resources/textures/achievement/window_btn_close.png`，点击可直接关闭销毁弹窗。

---

### 2.2 Daily 任务完成弹窗

* **触发时机**：关卡成功结算时，判断玩家今日通关总数达到每日目标限制（`dailyCompleted >= dailyGoal`）。
* **内容规格**：
  * **主标题**：在顶部标题条上显示 `DAILY GOAL DONE!`
  * **达成图示**：弹窗中央显示成就杯子图标 `icon_achievement.png`。
  * **达成提示**：下方渲染文本 `恭喜您完成今日目标！` (字号 `24`, 黄色 `#F4A261`)。
  * **获得奖励**：渲染金币图标并文字显示 `+50 金币`。
* **确定领奖按钮**：弹窗底部居中放置绿色按钮 `resources/textures/alert/main_btn_green.png`，渲染文字为 `common_txt_confirm.png` (或纯文本“确定”)，点击直接领取金币奖励并销毁弹窗。

---

### 2.3 图鉴获得弹窗

* **触发时机**：关卡顺利通关结算时，判断本关的关卡奖励描述包含掉落的未解锁零件（`level.collectionName != nil`）。
* **内容规格**：
  * **主标题**：显示 `NEW PART COLLECTED!`
  * **零件大图**：在弹窗正中央以 `240 × 240` 逻辑尺寸高亮渲染该零件的 AI 图鉴卡片资产。
  * **名称与描述**：下方渲染该部件的中文名称（如 `[稀有粘土齿轮]`）以及它所归属的建造机器与章节说明。
* **确定按钮**：底部放置绿色确认按钮 `main_btn_green.png`，点击将新收集到的零件写入本地存档，关闭弹窗。
